# HR Attrition Analysis

## Project Overview

The **HR Attrition Analysis** is a workforce analytics project designed to evaluate employee turnover patterns and identify key factors influencing attrition and job satisfaction.

This project demonstrates the ability to analyze employee data, build interactive dashboards, and derive actionable insights to support strategic HR decision-making using Excel.

---

## Key Objectives:

* **Attrition Analysis:** Identify patterns and trends in employee turnover.
* **Workforce Segmentation:** Analyze attrition across departments and job levels.
* **Employee Insights:** Evaluate factors such as job satisfaction and compensation.
* **Retention Strategy:** Provide insights to improve employee engagement and reduce attrition.

---

## Technical Stack & Environment

* **Tool:** Microsoft Excel
* **Techniques:** Data Cleaning, Pivot Tables, Charts, Dashboard Design

---

## Key Insights

* Higher attrition observed among junior-level employees
* Certain departments show consistently higher turnover rates
* Job satisfaction and salary play a significant role in attrition
* Workforce trends indicate areas requiring retention strategies
